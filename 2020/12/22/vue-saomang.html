Vue扫盲

2021年1月6日
| Vue
| 阅读
文章目录

前言：Vue是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue被设计为可以自底向上逐层应用。Vue的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue也完全能够为复杂的单页应用提供驱动。

Vue扫盲

一、起步

创建一个html，然后通过如下方式引入 Vue：

<!-- 开发环境版本，包含了有帮助的命令行警告 -->
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

或者

<!-- 生产环境版本，优化了尺寸和速度 -->
<script src="https://cdn.jsdelivr.net/npm/vue"></script>

或者

<!-- 先下载好静态资源放到工程目录中，再引入 -->
<script src="/js/vue/vue.min.js"></script>

创建

<div id="app">
	<p>{{ msg }}</p><!-- {{}}插值表达式-->
    <!-- 在被Vue接管的区域中使用Vue的系统指令呈现数据,这些指令就相当于是MVVM中的View这个角色 -->
</div>
<script type="text/javascript">
    // 实例化Vue对象 
    // vm :叫做MVVM中的 View Model
	var vm = new Vue({
		el:'#app', //表示当前vue对象接管id为app的div区域
		data:{
			msg:'Hello Vue!' // 相当于是MVVM中的Model这个角色
		}
	})
</script>

以上就成功创建了第一个 Vue 应用！Vue 将数据和 DOM 之间建立了关联，所有东西都是响应式的。打开控制台修改 msg 的值，你将看到上例相应地更新。

代码编写步骤：

  1. 定义html，引入vue.js
  2. 定义app div，此区域作为vue的接管区域
  3. 定义vue实例，接管app区域。
  4. 定义model（数据对象）
  5. VM完成在app中展示数据

<!DOCTYPE html>
<html lang="en">

<!--    <div>{{reverseMsg()}}</div>-->
    <div>{{reverseMsg}}</div>
    <div>{{copyMsg}}</div>
</div>
<script>
    var vm = new Vue({
        el:'#app',
        data:{
            msg:'hi chan!',
            copyMsg:null
        },
        /* 方法属性 不会缓存 每次调用方法都要执行一遍 即使msg不变的情况下 也要重新执行 */
        // methods:{
        //     reverseMsg(){
        //         return this.msg.split('').reverse().join('');
        //     }
        // },
        /* 计算属性 会缓存 msg不变的时候直接从缓存中拿值 msg变了的话 会重新执行 效率更高 */
        computed:{
            /* 两种写法等价 */
            /* 正常写法 这个reverseMsg方法可以当成一个普通的变量来使用 {{reverseMsg}} */
            reverseMsg:{
                get(){
                    return this.msg.split('').reverse().join('');
                },
                /* set方法 作用:在给reverseMsg赋值的时候 这个方法会被调用 */
                set(newVal){
                    // console.log("set:"+newVal);
                    this.copyMsg = newVal;
                }
            }
            /* 简写方式 这个reverseMsg方法可以当成一个普通的变量来使用 {{reverseMsg}} */
            /* computed计算属性的get方式用得多 一般这样简写的方式就够用了 */
            // reverseMsg(){
            //     return this.msg.split('').reverse().join('');
            // }
        }
    })
</script>
</body>
</html>

<!DOCTYPE html>

</div>
<script>
    var vm = new Vue({
        el: '#app',
        data: {
            /* 需求:如果用户手动修改了firstName或者lastName的值 那么这个fullName的值要跟着变 */
            firstName: '张',
            lastName: '三',
            fullName: '张三'
        },
        /* 这个案例只是为了演示watch侦听器 其实这个需求用computed更合适 watch更适合用于异步操作和耗时操作 */
        watch: {
            firstName(val) {
                this.fullName = val + ' ' + this.lastName;
            },
            lastName(val) {
                this.fullName = this.firstName + ' ' + val;
            }
        }
    })
</script>
</body>
</html>

十三、Vue 条件渲染

<!DOCTYPE html>
<html lang="en">

    <meta charset="UTF-8">
    <title>Title</title>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
</head>
<body>
<div id="app">
    <div v-show="isShow">{{msg}}</div><!--为false不显示时是用css的方式将元素隐藏 v-show不支持template-->
    <div v-if="isShow">{{msg}}</div><!--为false不显示时是元素移除-->
    <!-- 这里的template不是html的节点是不会渲染的 作用是统一控制下面三个标签是否显示 -->
    <template v-if="isShow">
        <p>{{msg}}</p>
        <div>hello chen</div>
        <span>blog.csy.ink</span>
    </template>
    <div v-else>迩东陈</div>
    性别:
    <div v-if="gender==0">男</div>
    <div v-else-if="gender==1">女</div>
    <div v-else>未知</div>
    <hr>
    <!-- 在vue里为了提高渲染效率 会对元素进行缓存 我们在用v-if的时候可能会出现一些缓存的情况 -->
    <template v-if="loginType=='username'">
        <!--切换到else的时候会有vue的缓存 label和input会复用 如果不想被复用 就加一个key 标记为表示不同的元素-->
        <label>用户名</label>
        <input type="text" placeholder="请输入用户名" key="username">
    </template>
    <template v-else>
        <label>用户名</label>
        <input type="text" placeholder="请输入邮箱地址" key="email">
    </template>
</div>
<script>
    var vm = new Vue({
        el: '#app',
        data: {
            isShow: false,
            msg: 'chan',
            gender:2,
            loginType: 'username'
        }
    })
</script>
</body>
</html>

    <table border="1">
        <tr>
            <td>下标</td>

            <td>作者</td>
        </tr>
        <!-- in或者of都可 不带下标时 -->
<!--        <tr v-for="book in books">-->
        <!-- 附带下标的写法 -->
        <tr v-for="(book,index) in books" v-bind:key="index">
            <td>{{index}}</td>
            <td>{{book.name}}</td>
            <td>{{book.author}}</td>
        </tr>
    </table>

    <hr>
<!--    遍历普通对象Obj-->
    <!-- 只显示对象的属性值 -->
    <div v-for="s in site">
        {{s}}
    </div>
    <hr>
    <!-- 显示对象的属性名和值和下标 -->
    <div v-for="(s,property,index) in site">
        {{index}}---{{property}}---{{s}}
    </div>

    <hr>
    <!-- 在这个循环渲染的时候有一个需要注意的就是"key"的问题 因为在循环渲染的时候 vue会认为这些div是同一个div 实际上并不相同 因此为了区分加key -->
    <!-- 这里的key要保证每个值不同 因此要使用一个变量 这里通过v-bind将key的值绑定为index -->
    <div v-for="(s,property,index) in site" v-bind:key="index">
        {{index}}---{{property}}---{{s}}
    </div>
</div>

    var vm = new Vue({

                },{
                    name:'水浒传',

            site:{

</script>

九九乘法表横纵拓展

<!DOCTYPE html>

<!-- 如果想自动将用户的输入值转为数值类型 可以给v-model添加number修饰符 -->
<!-- 如果只写成v-model="num" 那么num会被vue认为是字符串 .number将值转化成数值类型 实现99乘法表的横纵拓展 -->
<div id="app">
    <input type="number" v-model.number="num">
    <table border="1">
        <tr v-for="i in num">
            <!--j写在{{}}外面就是个普通的字符串,写在{{}}里面就当成变量来渲染-->
            <td v-for="j in i">{{j}}*{{i}}={{j*i}}</td>
        </tr>
    </table>
</div>
<script>
    var vm = new Vue({
        el:'#app',
        data:{
            num:9
        }
    })
</script>

十七、Vue 组件基础

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
</head>
<body>
<div id="app">
    <!--这里的chan标签就是我们下面自定义的组件 name就是组件中定义的props中的参数 hello就是VM中data中的参数值-->
    <chan v-bind:name="hi"></chan>
    <!--浏览器输出: hello chan...! -->
    <hr>

    <chen title="组件一"></chen>
    <chen title="组件二"></chen>
    <hr>

    <!--    <blog v-for="(blog,index) in blogs" v-bind:title="blog.title" v-bind:date="blog.date" v-bind:author="blog.author" v-bind:key="index"></blog>-->
    <!-- 简写 -->
    <!-- ---使用子组件blog 子组件中有点击事件sayhello 触发了父组件中的hello方法--- -->
    <blog @sayhello="hello" v-for="(blog,index) in blogs" :title="blog.title" :date="blog.date" :author="blog.author"
          :key="index">
        <!-- 插槽中的值 表示这个div将会在slot的位置显示 -->
        <div style="color: red">hello 插槽!</div>
    </blog>

    <hr>
    <!-- 动态组件 将这个组件标签与自定义的组件chen绑定 在vm的data里面指定dynamicComponent:chen 效果:显示chen组件的内容 -->
    <component v-bind:is="dynamicComponent"></component>
</div>
<script>
    /* 组件  第一个参数是组件的自定义的名字 第二个就是组件的内容 template表示这个组件它对应的标签是什么 */
    /* 相当于我自定义一个标签叫chan chan的值内容就是这个h1标签的内容 */
    /* 一个组件中只能有一个根元素 其他的东西只能在这个根元素里面 */
    Vue.component('chan', {
        props: ['name'],
        /*这里的name 就是上面的props中的参数*/
        template: '<h1>hello {{name}}!</h1>'
    });

    Vue.component('chen', {
        /* 在组件中定义data不能像VM中那样 要写成方法的格式 */
        data() {
            return {
                counter: 0
            }
        },
        props: ['title'],
        /* v-on:click里面是一个表达式 如果是多个表达式 就要封装成一个方法 这块写方法名和参数 */
        /* 一个组件中只能有一个元素 其他的东西只能在这个元素里面 */
        template: '<div><button v-on:click="counter++">hello {{counter}}!</button><div>{{title}}</div></div>'
    });

    Vue.component('blog', {
        props: ['title', 'date', 'author'],
        template: '<div>' +
            '<div>{{title}}</div>' +
            '<div>{{date}}---{{author}}</div>' +
            /* 给组件添加一个插槽 这个东西代表的值就是我们自定义的组件标签里面的内容 可有多个slot */
            '<slot></slot>' +
            /* ---监听子组件的事件: 效果相当于点击子组件里面的一个事件触发了父组件里面的方法 也就是vm中的hello方法--- */
            /* ---使用事件抛出一个值 测试抛出title 效果:点击事件触发时,将title参数console.log输出--- */
            '<button @click="$emit(\'sayhello\',title)">click</button>' +
            '</div>'
    });

    var vm = new Vue({
        el: '#app',
        methods: {
            /* ---父组件中的方法--- */
            hello(val) {
                console.log("hello chen!" + val);
            }
        },
        data: {
            /* 定义动态组件的绑定 */
            dynamicComponent:'chen',
            msg: 'chan!!!',
            hi: 'chan...',
            blogs: [
                {
                    title: '标题1',
                    date: '2021-1-13',
                    author: 'chen'
                }, {
                    title: '标题2',
                    date: '2021-1-11',
                    author: '迩东陈'
                }, {
                    title: '标题3',
                    date: '2021-1-12',
                    author: 'chan'
                }
            ]
        }
    })
</script>
</body>
</html>
