Spring Boot的文件上传

2020年12月21日
| Spring Boot
| 阅读
文章目录

前言：我们后端开发人员，一定手写过文件上传，如果最初是用SSM写的文件上传，而后使用Spring Boot了的话，那么我们一定会觉得在SSM 中会更加的麻烦，还是那句话Spring Boot就像个慈祥的保姆，什么都为我们设置了默认的配置，亦或像一位不那么独断专制的霸主，允许我们按照需求规定自己的配置，并且方便的一批。本文介绍Spring Boot中的文件上传。

Spring Boot中的文件上传

Spring Boot中的文件上传

首先，搭建普通的Maven工程，引入pom文件：

<parent>
    <groupId>org.springframework.boot</groupId>

</parent>

<dependencies>

        <groupId>org.springframework.boot</groupId>

1、Spring Boot中form表单的文件上传

首先，创建接口controller

package com.chan.controller;

import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RestController;
import org.springframework.web.multipart.MultipartFile;

import javax.servlet.http.HttpServletRequest;
import java.io.File;
import java.io.IOException;
import java.text.SimpleDateFormat;
import java.util.Date;
import java.util.UUID;

/**
 * @Classname UploadController
 * @Description TODO
 * @Date 2020/5/20 17:47
 * @Created by ChenSiYu
 */
@RestController
public class UploadController {

    //强行格式化日期一会做目录用的
    SimpleDateFormat format = new SimpleDateFormat("/yyyy/MM/dd/");

    /**
     * @return java.lang.String
     * @Description 上传文件的方法，注意参数中的multipartFile需要和前端的name属性的值一致
     * @Param [file, request]
     * @Date 2020/5/20 17:51
     * @Author ChenSiYu
     */
    @PostMapping("/upload")
    public String upload(MultipartFile multipartFile, HttpServletRequest request) {

        String format = this.format.format(new Date());

        //这里如果是公司可能需要搭建一个文件服务器,对应相应的路径,这里我们就简单点放在项目运行的路径下
        String realPath = request.getServletContext().getRealPath("img") + format;

        File file = new File(realPath);
        if (!file.exists()) {
            file.mkdirs();
        }

        String oldName = multipartFile.getOriginalFilename();
        String newName = UUID.randomUUID().toString() + oldName.substring(oldName.lastIndexOf("."));

        try {
            multipartFile.transferTo(new File(file, newName));
            String url = request.getScheme() + "://" + request.getServerName() + ":" + request.getServerPort() + "/img" + format + newName;
            return url;
        } catch (IOException e) {
            e.printStackTrace();
        }
        return "error";
    }

创建index.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<form action="/upload" method="post" enctype="multipart/form-data">
    <input type="file" name="multipartFile">
    <input type="submit" value="提交">
</form>
</body>
</html>

表单上传三要素：method="post"；enctype="multipart/form-data"；type="file"，另外需要注意的是接口方法中括号的参数与前端的name的值一致！！！

2、Spring Boot中的ajax方式的文件上传

这个时候，后端的接口代码和之前的一样，就不贴了^^。

前端新建index2.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <script src="jquery-2.1.0.min.js"></script>
</head>
<body>

<div id="result"></div>
<input type="file" id="multipartFile"><!--这里的id和接口方法中的参数一致!!!-->
<input type="button" value="上传" onclick="uploadFile()">
<script>
    function uploadFile() {
        var file = $("#multipartFile")[0].files[0];
        var formData = new FormData();
        formData.append("multipartFile",file);
        $.ajax({
            type:'post',
            url:'/upload',
            processData:false,
            contentType:false,
            data:formData,
            success:function (msg) {
                $("#result").html(msg);
            }
        })
    }
</script>

</body>
</html>

     */
    @PostMapping("/uploads")
    public String uploads(MultipartFile[] multipartFiles, HttpServletRequest request) {

        String format = this.format.format(new Date());

        //这里如果是公司可能需要搭建一个文件服务器,对应相应的路径,这里我们就简单点放在项目运行的路径下
        String realPath = request.getServletContext().getRealPath("img") + format;

        File file = new File(realPath);
        if (!file.exists()) {
            file.mkdirs();
        }

        for (MultipartFile multipartFile : multipartFiles) {
            String oldName = multipartFile.getOriginalFilename();

            }
        }

        return "success";
    }
}
