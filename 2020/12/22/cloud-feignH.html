前言：Spring Cloud之Feign(H版)。

一、Feign简介

Feign是一个声明式的伪Http客户端，它使得写Http客户端变得更简单。使用Feign，只需要创建一个接口并注解。它具有可插拔的注解特性，可使用Feign 注解和JAX-RS注解。Feign支持可插拔的编码器和解码器。Feign默认集成了Ribbon，并和Eureka结合，默认实现了负载均衡的效果。

简而言之：

  • Feign 采用的是基于接口的注解
  • Feign 整合了ribbon，具有负载均衡的能力
  • 整合了Hystrix，具有熔断的能力

二、准备工作

继续用上一节的工程， 启动 CSY-eureka，端口为20000；为了测试负载，我们启动 CSY-provider 和 CSY-provider2 构成集群，端口分别为20001、20003。

5、编写OrderFeign接口

案例是order对user信息的消费，体现服务远程间的消费场景。这里feign接口中的方法的返回值和参数与被消费者服务CSY-provider中的controller中的方法对应，映射路径对应，并且注意在feign中PathVariable注解和RequestParam注解必须指定参数的名字。PathVariable注解处理参数值在请求地址上的情况，RequestParam注解处理参数拼接到地址上或者参数在请求体中附带过来的情况。

/**
 * @Classname OrderFeign
 * @Description TODO
 * @Date 2020/9/15 22:13
 * @Created by ChenSiYu
 */
//配置当前接口为Feign的客户端,内部访问CSY-provider这个服务集群
@FeignClient(value = "CSY-PROVIDER")
public interface OrderFeign {

    @RequestMapping("/user/findUserList")
    List<TbUser> findUserList();

    @GetMapping("/user/findUser/{userId}")
    TbUser findUser(@PathVariable("userId") long userId);//注意在feign中PathVariable注解必须指定参数的名字,不指定会报错

    @RequestMapping("/hi")
    String home(@RequestParam(value = "name", defaultValue = "erdochan") String name);
}

6、编写controller

/**
 * @Classname OrderController
 * @Description TODO
 * @Date 2020/9/15 22:12
 * @Created by ChenSiYu
 */
@RestController
public class OrderController {

    @Autowired
    OrderFeign orderFeign;

    @RequestMapping("/order/getUsers")
    public List<TbUser> getUsers(){
        List<TbUser> userList = orderFeign.findUserList();
        return userList;
    }

    @RequestMapping("/order/getUser/{userId}")
    public TbUser getUser(@PathVariable("userId") long userId){
        TbUser user = orderFeign.findUser(userId);
        return user;
    }

    /**
     * @Description /order/hi/{name} --> /order/hi/csy --> @PathVariable;
     *              /order/hi/ --> /order/hi?name=csy&... --> @RequestParam.
     * @Param [name]
     * @return java.lang.String
     * @Date 2020/9/15 22:40
     * @Author ChenSiYu
     */
//    @RequestMapping("/order/hi/{name}")
    @RequestMapping("/order/hi")
    public String hi(@RequestParam("name") String name){
        String home = orderFeign.home(name);
        return home;
    }
}

四、使用feign的注解

在上面的案例中，我们在feign的接口使用的是Spring MVC中的注解，这里我们新建一个模块，使用feign的配置，在feign接口中使用feign的注解。

1、新建module子模块

命名为 CSY-feign-config。

2、创建properties文件

server.port=20005

spring.application.name=CSY-feign-config

eureka.client.serviceUrl.defaultZone=http://localhost:20000/eureka/

3、新建配置类

/**
 * @Classname OrderFeignConfig
 * @Description TODO
 * @Date 2020/9/15 23:20
 * @Created by ChenSiYu
 */
@Configuration
public class OrderFeignConfig {
    @Bean
    public Contract contract() {
        return new feign.Contract.Default();//默认的契约,也就是不再使用springmvc的注解
    }
}

4、编写feign接口

 * @Classname OrderFeign

public interface OrderFeign {

    @RequestLine("GET /user/findUserList")//两个参数以空格来区分,第一个是请求方式,第二个是请求地址

    @RequestLine("GET /user/findUser/{userId}")

//    @RequestLine("POST /save")

}
