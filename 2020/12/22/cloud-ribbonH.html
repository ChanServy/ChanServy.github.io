Spring Cloud之Ribbon(H版)

一、ribbon简介

Ribbon is a client side load balancer which gives you a lot of control over the behaviour of HTTP and TCP clients. Feign already uses Ribbon, so if you are using @FeignClient then this section also applies. —–摘自官网

ribbon是一个负载均衡客户端，可以很好的控制 http 和 tcp 的一些行为。Feign默认集成了ribbon。

ribbon 已经默认实现了这些配置bean：

  • IClientConfig ribbonClientConfig: DefaultClientConfigImpl
  • IRule ribbonRule: ZoneAvoidanceRule
  • IPing ribbonPing: NoOpPing
  • ServerList ribbonServerList: ConfigurationBasedServerList
  • ServerListFilter ribbonServerListFilter: ZonePreferenceServerListFilter
  • ILoadBalancer ribbonLoadBalancer: ZoneAwareLoadBalancer

二、准备工作

我们知道，eureka主要分为服务端和客户端，客户端向服务端注册，那客户端中，根据业务需求，又大致可以分为提供者和消费者。

因为ribbon一般是作为消费者的，因此在演示ribbon之前，我们首先建立两个服务提供者，为什么是两个呢？是因为ribbon有负载均衡的功能，想测试一波。很简单，其实是一样的，只不过变化一个端口号，两个提供者服务的服务名一致。

在父工程cloud的基础上，新建module，命名为CSY-provider。也是个空的maven子模块工程。

        PageHelper pageHelper = new PageHelper();
        Properties properties = new Properties();
        properties.setProperty("offsetAsPageNum","true");
        properties.setProperty("rowBoundsWithCount","true");
        properties.setProperty("reasonable","true");
        properties.setProperty("dialect","mysql");    //配置mysql数据库的方言

        return pageHelper;
    }
}

@RestController
@RequestMapping("/user")
public class TbUserController {

    private static final Logger LOGGER = LoggerFactory.getLogger(TbUserController.class);


    @Autowired
    TbUserService tbUserService;

    @RequestMapping("/findUserList")
    public List<TbUser> findUserList() {
        try {
            List<TbUser> users = tbUserService.findAll();
            LOGGER.info("------users------" + users);
            return users;
//            return new CommonResult(StatusCode.SUCCESS, "查询users成功", users);
        } catch (Exception e) {
            e.printStackTrace();
            return null;
//            return new CommonResult(StatusCode.ERROR, "查询users失败");
        }
    }

    @RequestMapping("/findUser/{userId}")
    public TbUser findUser(@PathVariable long userId){
        TbUser user = tbUserService.findById(userId);
        return user;
    }


}

public interface TbUserService {

    List<TbUser> findAll();

    TbUser findById(long userId);
}

@Service
public class TbUserServiceImpl implements TbUserService {

    @Autowired
    TbUserMapper tbUserMapper;

    @Override
    public List<TbUser> findAll() {
        return tbUserMapper.findAll();
    }

    @Override
    public TbUser findById(long userId) {
        return tbUserMapper.findById(userId);
    }

}

@Repository
@Mapper
public interface TbUserMapper {
    List<TbUser> findAll();

    TbUser findById(long userId);
}

        return new RestTemplate();
    }

}

负载均衡配置类

/**
 * @Classname RibbonConfig
 * @Description TODO
 * @Date 2020/9/14 19:25
 * @Created by ChenSiYu
 */
@Configuration
public class RibbonConfig {

    @Bean
    public IRule ribbonRule(){
        //线性轮循
        new RoundRobinRule();
        //可以重试的轮循
        new RetryRule();
        //根据运行情况来计算权重
        new WeightedResponseTimeRule();
        //过滤掉故障实例,选择请求数最小的实例
        new BestAvailableRule();
        //随机轮循
        return new RandomRule();
    }

}

controller

提供者服务的controller是有关user用户信息的，比如现在的场景是订单order的业务逻辑中需要得到user的信息，那么order即将作为消费者服务来消费user，也就是提供者服务。

/**
 * @Classname OrderController
 * @Description TODO
 * @Date 2020/9/14 19:31
 * @Created by ChenSiYu
 */
@RestController
public class OrderController {

    @Autowired
    private RestTemplate restTemplate; //一个专门请求rest风格url的对象

    @Autowired
    private LoadBalancerClient loadBalancerClient;//负载均衡的客户端

    @Autowired
    OrderService orderService;

    @RequestMapping("/order/{id}")
    public TbUser getUserInfo(@PathVariable long id) {
        TbUser user = restTemplate.getForObject("http://CSY-PROVIDER/user/findUser/" + id, TbUser.class);
        return user;
    }

    @RequestMapping("/test")
    public String choose() {
        int port = loadBalancerClient.choose("CSY-PROVIDER").getPort();
        System.err.println("----------------->"+port);
        return port + "";
    }

    @RequestMapping("/hi/{name}")
    public String hi(@PathVariable String name){
        String servicePort = orderService.getServicePort(name);
        return servicePort;
    }
}
